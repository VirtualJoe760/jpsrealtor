## 7. Priority Action Items

### P0 — CRITICAL (Today/Tomorrow)

1. **FIX SITEMAPS** — No sitemaps exist in GSC. Verify `/sitemap.xml`, `/sitemap/0.xml`, and `/sitemap/1.xml` resolve correctly on production. Then resubmit in GSC. Without a sitemap, Google cannot discover 5,000+ pages.

2. **FIX `/neighborhoods/palm-desert` 404** — This key city page is returning 404 in production. Google last crawled it Jan 10 and gave up. Check if the routing for `/neighborhoods/[cityId]` is broken in the Vercel deploy. This likely affects ALL city pages, not just Palm Desert.

3. **FIX `/insights` Soft 404** — Google crawls this page but classifies it as having no real content. The page likely renders a loading skeleton server-side (SSR) and fills in client-side, which Google sees as empty. Fix: ensure the insights page returns real content during SSR, not just a spinner.

### P1 — High (This Week)

4. **Fix brand query ranking** — "joseph sardella" at position 6.9 with 0% CTR is unacceptable. Add the brand name prominently to the homepage `<title>`, `<h1>`, and meta description. Fix the cannibalization with `/book-appointment` by adding `<link rel="canonical">` or removing competing meta.

5. **Fix mobile CTR gap** — Test title tag rendering on mobile search results. Titles may be truncating. Consider shorter titles (50-55 chars) for key pages.

6. **Fix performance regression** — TTFB doubled, Speed Index tripled. Profile the production build to identify:
   - New bundle bloat from recently added components (buy/sell pages, CMA, address autocomplete)
   - Database queries running on initial page load (community spotlight aggregation)
   - Consider lazy-loading the heavy components

### P2 — Medium (This Month)

7. **Internal linking overhaul** — Only 2 pages are indexed. Build internal links from the homepage → city pages → subdivision pages → buy/sell pages. Google can't discover what it can't follow.

8. **Content page CTR optimization** — "best-neighborhoods-for-young-family" has 49 impressions at position 10.5 with 0 clicks. Rewrite the meta title and description to be more compelling and action-oriented.

9. **Activate backlink building** — Still the biggest structural gap. Zero domain authority. Start with:
   - Local business directories (BBB, Chamber of Commerce)
   - Real estate association listings
   - Local news/community sites
   - Guest posts on Coachella Valley blogs

10. **Submit sitemap to Bing Webmaster Tools** — Free additional search visibility, no incremental effort once sitemaps are fixed.

### P3 — Low (Ongoing)

11. **Monitor the "exp realty palm springs" query** — 73 impressions, position 2, 0% CTR. This is an eXp Realty brand query where the user wants eXp corporate info, not an agent page. Not worth optimizing for, but don't lose the position either.

12. **Refresh decaying content** — The Indian Wells Tennis Tournament article lost all visibility. Update with 2026 dates/results.

13. **GBP optimization** — API access case was estimated for Apr 14. Follow up if not received.

## 8. Top 3 Takeaways

1. **The site is practically invisible to Google.** Only 2 of thousands of pages are indexed. The sitemap is gone, key city pages 404, and the insights hub is a soft 404. Nothing else matters until indexing is fixed.

2. **Performance regressed.** The new features (buy/sell pages, CMA engine, community spotlight) added latency. TTFB nearly doubled and Speed Index tripled. This needs profiling before it gets worse.

3. **Growth is happening despite everything.** Clicks up 67%, impressions up 36%. The site IS getting discovered organically — it just can't capitalize because most pages aren't indexed and the ones that are have poor CTR. Fixing indexing alone could 10x impressions.
