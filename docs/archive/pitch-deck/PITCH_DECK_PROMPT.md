## How to produce the deck

**Output:** A single PDF file, ~15 slides, presentation aspect ratio (16:9), landscape.

**Tone:** Confident, founder-led, plain-spoken. No hype words ("revolutionary," "game-changing," "disrupting"). Lead with facts and proof. Investors and brokers see hundreds of decks — this one wins on credibility.

**Design direction:**
- Dark, modern background with a soft green accent — matches the ChatRealty brand (logo is a chat bubble with a house roof; brand color is a saturated green/emerald)
- Sans-serif typography; large headlines (~60pt), generous whitespace
- One key idea per slide; minimal text
- Use real screenshots from the app where indicated (the slide content will tell you when — leave a placeholder if no screenshot is provided)
- Charts and diagrams only where they add information (don't decorate with stock graphs)

**Layout pattern per slide:** Slide number top-left, title large, supporting content centered or 2-column. Footer in muted text: "ChatRealty · chatrealty.io · 2026."

**Length:** Aim for ~15 slides. If a section can be compressed, compress it. A tight deck closes more rounds than a long one.

# SLIDE 9 — THE NETWORK EFFECT

**Title:** Every agent strengthens every other agent's SEO.

**Body (diagram + paragraph):**

ChatRealty is a hub-and-spoke domain network. Each agent's domain ranks independently in Google. Contextually relevant cross-links — through the Agent Directory, neighborhood pages, listing pages, and editorial content — flow authority between domains.

This is *not* a private blog network. Each site:
- Serves a real local business (a licensed agent's practice)
- Publishes unique, original content authored for that agent
- Maintains its own Google Search Console property
- Links to other agents only when there's contextual relevance (e.g., "Looking for an agent in Palm Springs? Visit Jane Doe's site.")

**The flywheel:** more agents → more domains with unique local content → more authority → all domains rank higher → more leads → agents scale spend → more revenue. Network value increases as agents are added.
